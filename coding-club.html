About the Club

The Coding Club at Manipal University is a vibrant community of tech enthusiasts dedicated to fostering programming skills and innovation. We believe in learning by doing and provide hands-on experience through workshops, hackathons, and collaborative projects.

Our Mission

To create a supportive environment where students can develop their programming skills, work on real-world projects, and prepare for successful careers in technology.

Our Activities

Weekly Workshops

Learn new technologies and programming concepts through hands-on workshops conducted by experts.

Hackathons

Participate in exciting 24-48 hour coding competitions and build innovative solutions.

Open Source Projects

Contribute to real-world projects and build your portfolio while helping the community.

Study Groups

Collaborative learning sessions for DSA, competitive programming, and interview preparation.

Tech Talks

Industry experts share insights about latest trends, career guidance, and emerging technologies.

Code Reviews

Peer code reviews to improve coding standards and learn best practices from fellow developers.
